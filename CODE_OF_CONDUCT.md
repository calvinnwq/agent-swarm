# Code of Conduct

Agent Swarm is a small public project. Keep participation practical, respectful,
and focused on making the tool better.

## Expected Behavior

- Be kind, direct, and constructive.
- Assume good intent, especially when discussing bugs or incomplete alpha behavior.
- Keep issues and PRs scoped to the project.
- Respect people's time, privacy, and boundaries.
- Redact secrets, private prompts, local paths, and customer or workplace details
  before posting examples.

## Unacceptable Behavior

- Harassment, threats, discrimination, or personal attacks.
- Sexualized, violent, or otherwise hostile comments.
- Publishing private information without permission.
- Repeated off-topic comments, spam, or bad-faith argument.
- Using issues, PRs, or discussions to pressure maintainers for private support.

## Enforcement

Calvin is the project maintainer and may edit, hide, lock, or delete comments;
close issues or PRs; block users; or report abuse to GitHub when needed.

If you see a conduct problem, report it privately when possible. If private
reporting is unavailable, keep any public report minimal and avoid repeating
harmful content.

## Scope

This code applies to participation in the Agent Swarm repository, issues, pull
requests, releases, and related project channels.
